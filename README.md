# Chat
## 1. Linux下的聊天程序（P2P）
    QT = 5.14.2  
    多路复用IO模型用的epoll  
    无第三方服务器打孔，只能用在局域网下  
   
## 2. 使用方法
    git clone https://github.com/YeFeng1993/Chat
    用QT打开工程，直接编译运行  
    ![](https://github.com/YeFeng1993/images/blob/main/example.png)
    
    
## 3. 需要添加的功能
    文件传输
    群聊
    聊天记录保存
    UI美化
